# kangbenpro
康本项目手机端，尝试接入钉钉
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
在运行之前需要修改env.development中的参数配置
同时第一次需要运行在钉钉环境中，为了获取AccessToken和UserId
然后想要在网页上调试的话,需要手动配置上面2个属性
```
